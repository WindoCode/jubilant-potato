# GvG Strategy Board – Where Winds Meet

Interactive tactical planning board for Guild vs Guild coordination, map drawing, and strategy visualization.
https://windocode.github.io/jubilant-potato/ 

---

# Features

## Tactical Drawing Tools
- Freehand drawing directly on the map
- Adjustable brush size
- Multi-color tactical overlays
- Eraser support
- Undo last action
- Clear entire board

## Arrows
- Create directional movement arrows
- Attack path visualization
- Push / flank route planning
- Adjustable arrow thickness
- Team-colored routing

## Marker Placement
Place tactical markers anywhere on the map.

### Available Marker Types
- ⚔ Sword
- 🛡 Shield
- 💚 Healer
- 💀 Skull
- ★ Star
- ✓ Check
- ❌ X
- Tower
- Tree 
- Goose

### Marker Features
- Drag and reposition
- Right-click context menu
- Add custom labels
- Change marker color
- Delete individual markers

# Controls

## Mouse Controls

| Action | Function |
|---|---|
| Left Click + Drag | Draw / Create Arrow |
| Mouse Wheel | Zoom |
| Middle Mouse Drag | Pan |
| Right Click Marker | Open Context Menu |

# Keyboard Shortcuts

| Key | Action |
|---|---|
| D | Draw Mode |
| A | Arrow Mode |
| M | Marker Mode |
| P | Pan Mode |
| R | Red Team |
| B | Blue Team |
| Ctrl + Z | Undo |
| + / = | Zoom In |
| - | Zoom Out |
| 0 | Reset View |
| Escape | Close Menus |
